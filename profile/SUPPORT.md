# Support

## Need Help?

If you need support for any of our projects, please follow these steps:

1. **Check the Documentation:** Make sure to read the README and any (if) available documentation in the repository.
2. **Search Issues:** See if someone else has already asked your question by searching the issues (both open and closed) in the relevant repository.
3. **Open a New Issue:** If you can't find an answer, open a new issue. Please provide as much detail as possible.

## Community Support

You can also reach out to our community for help:
- [Discord / Slack]
- [Stack Overflow using our specific tag]
- [GitHub Discussions]
